Application Programming Interface

Di seguito una serie di metodi per accedere ai dataset utilizzati all'interno di queste pagine

Andamenti per Varco

URL

monitoraggioareac.trecode.webfactional.com/dati/andamenti_areac.json

Campi

transiti_vds, veicoli_autorizzati, transiti_residenti, transiti, transiti_autorizzati,veicoli_paganti, veicoli_residenti, transiti_ecologici, veicoli, veicoli_vds, transiti_paganti, giorno, transiti_stimati,veicoli_ecologici

id_varco, giorno, transiti, veicoli, transiti_corretti, percentuale, transiti_c (fascia di correzzione)

Descrizione

Andamenti totali giornalieri degli ingressi ad Area C suddivisi per varco di accesso durante le fasce orarie con provvediemtno attivo
Andamenti giornalieri per macroclassi

URL

monitoraggioareac.trecode.webfactional.com/dati/andamento_giorno_classi.json

Campi

transiti_vds, veicoli_autorizzati, transiti_residenti, transiti, transiti_autorizzati,veicoli_paganti, veicoli_residenti, transiti_ecologici, veicoli, veicoli_vds, transiti_paganti, giorno, transiti_stimati,veicoli_ecologici

Descrizione

Veicoli e Transiti totali giornalieri suddiviso in macroclassi.
Anagrafica Varchi

URL

monitoraggioareac.trecode.webfactional.com/dati/varchi.json

Campi

lat, lon, tipo, id, nome

Descrizione

Varchi georefenziati appartenenti alla ZTL Area C
Andamenti orario 2016

URL

monitoraggioareac.trecode.webfactional.com/dati/andamento_orairo_2016.json

Campi

transiti_areac, transiti_non_areac, giorno, ora

Descrizione

Totale dei transiti giornalieri in ingresso alla ZTL durante le fasce orarie con provvedimento attivo e non
Andamenti dei funzionamenti

URL

monitoraggioareac.trecode.webfactional.com/andamento_giorno_funzionamenti.json

Campi

guasti, introito_mancato, transiti, transiti_corretti, transiti_rilevati, pagamento_transiti, varchi_funzionanti, veicoli, giorno, malfunzionamenti

Descrizione

Andamento degli ingressi per giorno con dettaglio dello stato di funzioanmento dei vrchi
